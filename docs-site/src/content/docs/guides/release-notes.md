title: Release Notes
description: Technical-preview release wording for Sourceright.
---

Release notes should describe implemented local triage, fixture-backed
benchmarks, schema contracts, and preview workflows without implying production
or externally benchmarked maturity.

Use "technical preview", "pilot-ready workflow", and "fixture-backed regression
benchmark". Avoid unsupported production, legal-compliance, or public-ranking
claims.
