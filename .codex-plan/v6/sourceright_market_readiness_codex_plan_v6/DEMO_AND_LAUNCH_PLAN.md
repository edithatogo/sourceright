# Demo and launch plan

## Demo goal

Show value in under three minutes without live provider calls.

## Static GitHub Pages demo

Use for:
- product explanation;
- sample report cards;
- no API keys;
- no provider calls;
- browser-only rendering of sample JSON.

Required:
- link from README;
- link from docs homepage;
- sample report JSON;
- sample journal screening JSON;
- status badges or “technical preview” label.

## Streamlit demo

Use for:
- uploading sample/local artifacts;
- exploring report JSON;
- review queue inspection;
- export readiness;
- plugin status matrix visualization.

Required:
- no API keys;
- no remote calls by default;
- clear “sample/demo only” text;
- README with local command:
  - `streamlit run streamlit_app/app.py`
