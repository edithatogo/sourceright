# Market-readiness checklist

## Recommended market position

Use:

> Sourceright is an open, auditable reference-verification and citation-integrity toolkit for scholarly and legal workflows. It is currently a technical preview suitable for developer evaluation and pilot conversations.

Avoid:

> SOTA citation checker.
> Production-ready journal onboarding platform.
> Court filing compliance system.
> AI detector.

## Ready for

- technical preview announcement;
- developer/library/publisher feedback;
- friendly journal/preprint pilot conversations;
- university repository metadata pilot conversations;
- MCP/citation-integrity demos;
- academic demo paper or workshop submission.

## Not yet ready for

- procurement-facing institutional sales;
- production claims for journals/courts;
- legally consequential filing checks;
- SOTA benchmark claims;
- broad end-user marketing without demos/docs.

## Public launch requirements

Before a public technical-preview launch:

- README has a “Who this is for” section.
- README has a “What it does / does not do” section.
- README links to public demos.
- Docs include a quickstart with commands and expected outputs.
- Benchmarks page explains runner status, tasks, baselines, and limits.
- Plugin/provider status is clearly labelled.
- Limitations page exists.
- At least one case study is documented.
- Rust API docs are materially improved.
- Release notes state preview limitations.
- Issue templates exist for bug reports, pilot feedback, provider requests, and docs gaps.

## Pilot partner pitch

Target:
- journal technical editors;
- university libraries and repository teams;
- preprint moderators;
- research-integrity groups;
- legal-tech researchers.

Pitch:
- “We help turn messy reference lists into clean, audited CSL and export artifacts.”
- “We keep provider evidence in a sidecar instead of silently overwriting references.”
- “We route uncertainty to review rather than pretending every lookup is certain.”
- “The current preview is deterministic and fixture-backed; live provider expansion is opt-in.”
