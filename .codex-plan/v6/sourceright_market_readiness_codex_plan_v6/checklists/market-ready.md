# Market-ready checklist

## Technical-preview launch

- [ ] README has “who this is for”.
- [ ] README has “what it does / does not do”.
- [ ] README links to demos.
- [ ] README states benchmark status accurately.
- [ ] Docs quickstart exists.
- [ ] At least one worked example exists.
- [ ] Benchmarks page explains task status and limitations.
- [ ] Plugin/provider status matrix exists.
- [ ] Demos use sample data only.
- [ ] Limitations page exists.
- [ ] Rust API docs improved.
- [ ] Release notes say technical preview.
- [ ] Pilot feedback issue template exists.
- [ ] No production/SOTA/legal compliance overclaims.
