# How to use this plan pack with Codex

1. Clone or update Sourceright locally.
2. Create a branch.
3. Unzip this plan pack outside the repo.
4. Copy only `AGENTS.md.template` into the repo as `AGENTS.md`.
5. Start Codex from the repo root.
6. Paste `codex/prompts/00-inspect-current-repo.md`.
7. Review Codex's plan.
8. Paste `codex/prompts/01-baseline-checks.md`.
9. Continue one slice at a time.

Do not bulk-copy this pack into the repo. The pack is guidance and prompts, not a patch.
