# Provider Strategy

Academic verification starts with Crossref and DOI resolution. Additional providers should be added only when they improve verification or enrichment quality.

Candidate academic providers:

- Crossref.
- DOI resolution.
- DataCite.
- OpenAlex.
- PubMed/NCBI.
- ORCID.

Provider results are candidates with provenance. They must not silently overwrite canonical references.
