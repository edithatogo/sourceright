# Sourceright (Qwen Code extension)

Registers `sourceright mcp` as an MCP server for Qwen Code.

## Install

```bash
qwen extensions install https://github.com/edithatogo/sourceright
# or local path to this directory after cloning:
qwen extensions install /path/to/sourceright/extensions/qwen-sourceright
```

Restart Qwen Code after install.
