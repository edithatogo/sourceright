# Synthetic Case Studies

This directory contains fake-but-realistic fixtures for reference handling and
review workflows.

The cases are intentionally conservative and synthetic. They are meant to show:

- biomedical manuscript handling with a retraction and version warning
- preprint handling where a version of record exists
- legal filing handling where a citation is ambiguous and should not be inferred

Each case includes:

- `input.md` - the synthetic source text or intake fragment
- `expected-artifact-notes.md` - the expected review notes and artifact behavior

The manifest in this directory is a lightweight inventory for the cases. These
fixtures are not tied to live providers and should not be treated as real-world
citations.
