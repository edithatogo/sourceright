# Claim Source Provenance Spec

## Goal

Roadmap the broader claim/source/provenance layer after reference verification is reliable.

## Scope

- Claim extraction.
- Claim-to-source and quote-to-source linkage.
- Evidence graphs.
- Source quality and provenance scoring.
- Document-level provenance reports.

## Boundaries

This track does not block v1 reference verification. It builds on verified references, citation reconciliation, and sidecar provenance.
