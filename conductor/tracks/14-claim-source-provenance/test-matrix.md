# Claim Source Provenance Test Matrix

| Scenario | Expected result |
| --- | --- |
| Claim extraction | Claim is linked to local document span. |
| Citation-backed claim | Claim links to a verified reference and citation occurrence. |
| Quote verification | Source quote evidence and mismatch diagnostics are recorded. |
| Unsupported claim | Manual-review or unsupported-source diagnostic is emitted. |
