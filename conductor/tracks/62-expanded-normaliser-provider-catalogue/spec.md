# Expanded Normaliser Provider Catalogue Spec

## Goal

Assess and track additional normalisers/providers so the roadmap covers the
databases Sourceright users will expect without overclaiming support.

## Provider Families

- Existing core/public: Crossref, DataCite, OpenAlex, PubMed/NCBI, ORCID,
  Unpaywall, OpenCitations, arXiv, Europe PMC.
- Licensed: Dimensions, Scopus, Web of Science.
- Search/index: Google Scholar is assessment-only unless terms and technical
  constraints allow a compliant path.
- Economics/social science: RePEc, EconLit decision path, SSRN, NBER, IDEAS.
- Grey literature and repositories: Zenodo, OSF, Figshare, Dataverse,
  institutional repositories, policy/working-paper repositories.
- Biomedical/preprint: Europe PMC, bioRxiv/medRxiv, clinical trial registries.

Each candidate needs status, data contract, access model, fixtures, live-smoke
policy, and deferral rationale.
