# Legacy Audit Spec

## Goal

Audit `legacy/humanizer-next` as source material for reference-verification behavior, fixtures, and discarded approaches.

## Scope

- Inventory citation/reference code, docs, and tests.
- Map inputs, outputs, formats, and assumptions.
- Identify reusable fixtures and behavior.
- Record what should be ported, replaced, or deleted.

## Boundaries

Legacy code remains non-production until explicitly ported into Rust.
