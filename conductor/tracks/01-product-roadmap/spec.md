# Product Roadmap Spec

## Goal

Keep the product centred on reference verification while preserving clear roadmap space for legal citations and claim/source/provenance work.

## Scope

- Define the academic-reference v1 workflow.
- Define legal citations as a later model/provider track.
- Define claim/source/provenance as a later layer above verified references.

## Interfaces

- Roadmap docs in `conductor/`.
- Public docs pages explaining workflow stages and future tracks.
