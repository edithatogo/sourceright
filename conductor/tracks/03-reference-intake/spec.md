# Reference Intake Spec

## Goal

Extract candidate reference-list entries and in-text citations from user-provided material.

## Scope

- Start with pasted text, plain text, Markdown, and DOCX.
- Progress to PDF text extraction.
- Add OCR/scanned documents after text extraction is stable.
- Preserve original extracted text for audit and manual review.

## Outputs

- Candidate reference records.
- Candidate in-text citation records.
- Extraction provenance and source spans where available.
