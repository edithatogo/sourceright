# Public Repo Infrastructure Spec

## Goal

Make Sourceright a public, professionally governed GitHub project at `edithatogo/sourceright`.

## Scope

- Public repository setup, MIT/Apache-2.0 licensing, GitHub metadata, issue/PR templates, and contribution docs.
- GitHub Actions for Rust CI, security checks, Pages, and tagged releases.
- GitHub Pages documentation site.
- Publishing plan for the CLI and MCP server mode.

## Interfaces

- Repository URL: `https://github.com/edithatogo/sourceright`.
- Pages URL: `https://edithatogo.github.io/sourceright/`.
- Release channel: GitHub Releases and crates.io.

## Boundaries

This track does not implement reference extraction or verification behavior.
