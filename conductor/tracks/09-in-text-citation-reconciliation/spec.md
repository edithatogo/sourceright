# In-Text Citation Reconciliation Spec

## Goal

Match body citations to reference-list entries and report document-level citation integrity issues.

## Scope

- Author-date, numeric, and footnote-like citation detection where practical.
- Missing references, uncited references, duplicate citations, numbering/order issues, and ambiguous matches.

## Outputs

- Citation reconciliation report.
- Sidecar links between in-text citations and CSL reference IDs.
