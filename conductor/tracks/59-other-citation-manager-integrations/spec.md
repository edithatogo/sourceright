# Other Citation-Manager Integrations Spec

## Goal

Cover citation-manager support beyond Zotero, with EndNote reference checking
as the first hard requirement and explicit decisions for Mendeley, Paperpile,
JabRef, RefWorks, Citavi, and CSL-compatible file workflows.

## Contract

- EndNote support starts as ENW/RIS handoff and reference-checking proof.
- Other managers get either an adapter plan, export/import proof, or explicit
  deferral.
- No manager is called supported without fixture-backed import/export or live
  smoke evidence.
