# Licensed BYO-Key Provider Adapters Spec

Define the licensed-provider path for Dimensions, Scopus, and Web of Science.
These adapters require bring-your-own-key configuration, privacy-aware caching,
clear skip diagnostics, and no default CI live calls.

The track may implement adapters only where API access is available. Otherwise
it must leave a precise deferral contract and fixtures based on permitted sample
responses.
