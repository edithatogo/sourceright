# Plugin Packaging And Supply-Chain Maturity Test Matrix

| Scenario | Acceptance |
| --- | --- |
| No submodules | Docs state submodules are not the default and list split criteria. |
| Evidence ledger | Plugin status requires fixtures, docs, install path, version, and compatibility. |
| Provenance | Installable plugins define signing/pinning/SBOM expectations. |
| Sandbox | Network/auth/cache permissions are explicit per plugin. |
| Review loop | `$conductor-review` runs and local fixes are applied. |
