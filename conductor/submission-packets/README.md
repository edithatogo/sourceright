# Submission Packets

This folder holds the machine-readable packet families for external
submissions. The packet manifest lives in `manifest.json`.

Each packet defines the local gates, blockers, and approval boundary for a
single external surface family. External issues, pull requests, and directory
listings require explicit approval before they can be created.

## Packet Families

- Registry and directory submissions
- Citation-manager submissions
- Journal-platform submissions
- arXiv upstream submissions
- AI client extension submissions
- VS Code and Open VSX submissions
- Agent/workflow submissions
