# GitHub Pages Demo Checklist

- Open [index.html](./index.html) and confirm the sample report card renders.
- Verify the metrics row, open-issues list, and journal screening summary.
- Confirm the page copy states that no live providers or API keys are used.
- Check that the sample data loads from `sample/reference-report.json` and
  `sample/journal-screening.json`.
- If the page is hosted on GitHub Pages, confirm the relative asset links still
  resolve correctly.
