# Streamlit Demo Checklist

- Run `pip install -r streamlit_app/requirements.txt`.
- Start the app with `streamlit run streamlit_app/app.py`.
- Confirm the title, metric cards, open issues table, and journal screening
  summary render from the sample JSON files.
- Verify the page copy states that the demo uses synthetic data only and does
  not call live providers.
- Confirm the optional report-card explanation is visible and matches the
  sample metrics.
